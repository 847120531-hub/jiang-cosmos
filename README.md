## 开启网站（GitHub Pages）

1. 打开 https://github.com/847120531-hub/jiang-cosmos/settings/pages
2. **Source** 选 **Deploy from a branch**（从分支部署）
3. **Branch** 选 **main**，文件夹选 **/ (root)**
4. 点 **Save**（保存）
5. 等 1～2 分钟
